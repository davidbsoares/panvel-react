## ⌨ Como executar o projeto

```bash
# Clonar o repositório
git clone https://github.com/guilhermecapitao/proffy-discovery

# Entrar no diretório
cd proffy-discovery

# Baixar as dependências
npm i

# Executar o servidor
npm run server

# Executar o projeto
npm start
```

Feito isso, abra o seu navegador e acesse `http://localhost:3000/`
